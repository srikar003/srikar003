## 🎯 Featured Build — Recruiting Screener

> Recruiter-first screening app that helps streamline job intake, candidate tracking, interview review, transcription, and evaluation.

Recruiting Screener is a full-stack AI-powered recruiting workflow application built to help recruiters move faster during the initial screening process. Recruiters can log in, paste a job description, let the system infer the likely role and level, generate screening questions, add multiple candidates, upload interview recordings, and review saved transcripts, scores, and evaluations.

This app provides value by reducing manual screening effort, keeping candidate information organized, helping recruiters create more consistent interview questions, and making interview review easier through stored transcripts, scoring, and evaluation history.

**Key capabilities:**

- Recruiter login using name and email
- Job description intake for screening workflow creation
- Role and level inference from the job description
- Recruiter override support when the inferred role or level needs adjustment
- AI-assisted screening question generation based on role and seniority level
- Candidate tracking under each job
- Interview recording upload support for candidate review
- Transcript, score, evaluation, and recording metadata storage
- Optional S3 support for storing interview recordings when configured
- Local storage fallback for development use

**Stack:** React · FastAPI · Python · PostgreSQL · AWS S3 · AI Inference · Transcription · Scoring

## 🧠 Featured Build — AI DevOps Agent

> AI-powered pull request reviewer for CI, security, and code quality automation.

Built a multi-agent AI DevOps workflow that reviews GitHub pull requests, runs CI/security checks, ranks findings, and prepares structured review comments for developers.

**What it does:**

- Fetches pull request metadata and changed files
- Runs CI checks, linting, and dependency/security scans
- Uses LangGraph to orchestrate multi-step review agents
- Ranks issues by severity and developer impact
- Generates clear review comments with remediation suggestions

**Stack:** Python · LangGraph · FastAPI · GitHub API · MCP-style services · CI/CD · Security Scanning
